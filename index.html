      How to setup Firebase Authentication with Vue Application
      Introduction Firebase provides a SDK to easily integrate the auth service into our web app. The user credentials are collected using the web app. The credentials are sent to firebase via SDK. Firebase takes care of storing the user information on registration and authentication. Goals We want to implement an vue app with authentication. User should be able to view only some sections our web app without authentication. If an unauthenticated user tries to view app sections which require authentication he will be redirected to the login page.
